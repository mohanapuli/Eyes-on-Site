Eyes On Site:
Steps to execute the project:
1. Open command prompt in the project directory.
2. Use Command "streamlit run app.py" to execute the project.
3. In frontend there will be option to choose the image/video/webcam..
4. select one of them and upload.
5. Click on detect object button.
6. It will detect the persons who are not wearing PPE kit and it will give alert sound.
